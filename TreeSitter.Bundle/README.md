# TreeSitter.Bundle

Language bundle for C# [Tree-sitter][] [NuGet][] package.

Don't use this package if you:

1. Want to compile your own grammar binaries.
2. Doesn't trust the prebuilt binaries.
3. Only want to use the C# bindings (core [TreeSitter][] package).
